Step
1)type system -config-network
2)the network configuration window appear with devices tab 
select by default
3)select the hardware tab to configure the ethernet card
4)select the ethernet card that you wish to configure
5)click edit button
6)A network adaptor configuration dialogue box appear.
7) click on drop-down menu of the device option
8) Select any device name foe ethernet card
9) click OK.
10) Again the network configurstion window appear ,select
file menu and save your changes.
11) Again select the file menu and click on quit from network
configuration window.
